# 项目代码仓库

## 项目简介
这是一个包含多种技术演示的项目，包括网页开发、Python编程和Java示例代码。

## 目录结构
```
.
├── web/           # 网页相关文件
├── python/        # Python相关文件
├── java/          # Java相关文件
└── README.md      # 项目说明文档
```

## 包含的文件

### 网页文件 (web/)
- `index.html` - 个人网页
- `qiuhuimin_profile.html` - 邱慧敏个人简介网页

### Python文件 (python/)
- `final_crawler.py` - Python爬虫脚本
- `file_operation_example.py` - 文件操作示例
- `python_learning.py` - Python学习基础示例
- `Python-test.py` - Python测试文件

### Java文件 (java/)
- `HelloWorld.java` - Java Hello World示例

## 项目特点
- 清晰的文件分类结构
- 包含多种技术演示
- 适合学习和参考
- 持续更新中

## 使用说明
1. 网页文件：直接在浏览器中打开
2. Python文件：使用Python 3运行
3. Java文件：使用Java编译器和运行环境

## 版本控制
本项目使用Git进行版本控制，所有更改都会被记录和跟踪。
